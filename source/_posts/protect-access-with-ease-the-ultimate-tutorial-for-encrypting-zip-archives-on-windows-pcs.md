##  Password Protect a ZIP Using 7-Zip

[7-Zip](https://www.7-zip.org/) is a free and open-source app that you can use to create and extract files from various archive formats, including ZIP. With this app, you choose the files to add to an archive, set the password, and let the app secure your archive's contents.

 Next, [open a File Explorer window](https://youtube-web.techidaily.com/ed-2024-approved-streamers-financial-health-check-in-youtubers/) and locate the files to add to a ZIP archive.
